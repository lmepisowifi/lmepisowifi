Unified SSID

Pick one active 2.4 GHz interface and one active 5 GHz interface. Their network name (SSID) and security / encryption are kept identical — changing one band automatically updates the other. Enabling copies the 2.4 GHz settings onto the selected 5 GHz interface.

Coin Slot (NodeMCU)
Coin Slot WiFi Sync

Tell the portal which interface the coin-slot NodeMCU connects to. When that interface's SSID or password changes, the portal first sends the new credentials to the NodeMCU and waits for its acknowledgement before switching the radio — so the coin slot is never stranded offline.

⚠️ The coin slot is not enabled on this router, so WiFi sync has no effect until you enable it in the coin-slot settings.
